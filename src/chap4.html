# Animations Jusqu'à présent, nous avons créer des images statiques. Pour faire des animations, il suffit de créer de nombreuses images statiques, plusieurs fois par secondes. L'idéal étant d'atteindre les 60 images par seconde. Pour cela, nous allons utiliser la fonction Javascript suivantes : __window.requestAnimationFrame( drawingFunction )__. Cette dernière demande au système d'exécuter la fonction __drawingFunction()__ dès le prochain rafraichissement de l'écran. Il faut noter deux choses : * __drawingFunction()__ reçoit un argument unique qui correspond au nombre de millisecondes écoulées depuis le 1er janvier 1970. * __drawingFunction()__ n'est appelée que si le canvas est visible à l'écran.
