Il est possible de créer des polygônes de n'importe quelle forme. Ces derniers seront toujours constitués de triangles dont les sommets seront pris dans un tableau. Mais à partir de cet unique tableau, il existe plusieurs façons de dessiner nos triangles.

Regardons de près les arguments de la fonction drawArrays( mode, first, count ).

  • first : index du premier point. Dans un tableau, le premier point a l'index 0.
  • count : nombre de points à lire.
  • mode : indique ce qu'il faut faire avec ces points
    • gl.POINTS : afficher seulement un pixel pour chaque point. (on verra plus tard comment définir la taille de ces "pixels").
    • gl.LINE_STRIP : tracer une ligne qui passe par chaque point.
    • gl.LINE_LOOP : tracer une ligne qui passe par chaque point, mais ajoute un segment entre le dernier et le premier point, formant ainsi une boucle.
    • gl.LINES : Prend deux points et trace un segment. Puis on continue ainsi tant qu'il reste au moins deux points. Cela permet de créer des lignes disjointes.
    • gl.TRIANGLE_STRIP : pour les points [A, B, C, D, E], on dessinera 3 triangles : ABC, BDC, CDE.
    • gl.TRIANGLE_FAN : pour les points [A, B, C, D, E], on dessinera 3 triangles : ABC, ACD, ADE. On penser à un éventail.
    • gl.TRAIANGLES : dessine des triangles qui peuvent être disjoints. pour les points [A, B, C, D, E, F], on dessinera 2 triangles : ABC, DEF.

Et, bien sûr, rien ne vous empêche d'enchaîner les appels :

    gl.drawArrays(gl.TRIANGLE_FAN, 0, 7);
    gl.drawArrays(gl.TRIANGLES, 8, 38);
    gl.drawArrays(gl.LINE_LOOP, 45, 53);
    ...
